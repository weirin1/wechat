## Requirement

1. PHP >= 7.0
2. **[Composer](https://getcomposer.org/)**

## Installation

```shell
$ composer require "weirin/wechat:~1.0" -vvv
```

## Usage

基本使用（以服务端为例）:

```php
<?php

use Weirin\Jwt;


```

更多请参考 [https://www.weirin.com/](https://www.weirin.com/)。
